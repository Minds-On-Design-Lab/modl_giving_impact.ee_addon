h1. MODL Giving Impact

h2. Overview

An ExpressionEngine 2.x module to interact with Giving Impact(TM).

*Developed By:* Minds On Design Lab - http://mod-lab.com
*Version:* 1.0

h2. Installation

Install in system/expressionengine/third_party/modl_giving_impact

h2. Usage

h3. Campaigns

Returns all active campaigns

<code>
{exp:giving_impact:campaigns}
	{gi_title}
	{gi_description}
	{gi_token}
	{gi_donation_url}
	{gi_share_url}
	{gi_donation_target}
	{gi_donation_total}
	{gi_image_url}
{/exp:giving_impact:campaigns}
</code>
