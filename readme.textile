h1. MODL Giving Impact

h2. Overview

An ExpressionEngine 2.x module to interact with Giving Impact(TM).

h2. Installation

Install in system/expressionengine/third_party/modl_giving_impact

h2. Usage

h3. Campaigns

Returns all active campaigns

<pre>
{exp:modl_giving_impact:campaigns}
	{gi_title}
	{gi_description}
	{gi_token}
	{gi_donation_url}
	{gi_share_url}
	{gi_donation_target}
	{gi_donation_total}
	{gi_image_url}
{/exp:modl_giving_impact:campaigns}
</pre>

h3. Single Campaign

Returns data for campaign with provided token

<pre>
{exp:giving_impact:campaigns token="[unique-campaign-token]"}
	{gi_title}
	{gi_description}
	{gi_token}
	{gi_donation_url}
	{gi_share_url}
	{gi_donation_target}
	{gi_donation_total}
	{gi_image_url}
{/exp:giving_impact:single-campaign}
</pre>

h3. Campaign Giving Opportunities

Returns all giving opportunities within campaign with provided token

<pre>
{exp:giving_impact:campaign_giving_opportunities token="[unique-campaign-token]"}
	{gi_title}
	{gi_description}
	{gi_token}
	{gi_donation_url}
	{gi_share_url}
	{gi_donation_target}
	{gi_donation_total}
	{gi_image_url}
{/exp:giving_impact:campaign_giving_opportunities}
</pre>

h4. Optional Parameters

* sort_by="[tag-name]"
* numeric="yes"
* backwards="yes"
* limit="[number]"
* random="[number]"

h3. Single Giving Opportunity

Returns data for a Giving Opportunity with provided token

<pre>
{exp:giving_impact:single_giving_opportunity token="[unique-giving-opp-token]"}
	{gi_title}
	{gi_description}
	{gi_token}
	{gi_donation_url}
	{gi_share_url}
	{gi_donation_target}
	{gi_donation_total}
	{gi_image_url}
{/exp:giving_impact:single_giving_opportunity}
</pre>

h3. Single Giving Opportunity Donation Log

Returns donation log data for a specific Giving Opportunity with provided token

<pre>
{exp:giving_impact:single_giving_opportunity_donation_log token="[unique-giving-opp-token]"}
	{gi_log_date}
	{gi_log_amount}
	{gi_log_name}
{/exp:giving_impact:single_giving_opportunity_donation_log}
</pre>
